📝 Cómo Editar & Agregar Preguntas

📂 Archivo Principal:

us_civics_spanish.json

Este es el archivo que debe contener todas tus preguntas en español.

📋 Formato de Pregunta:

{
  "id": 1,
  "block": "GOBIERNO",
  "question": "¿Tu pregunta?",
  "correct": "A",
  "options": {
    "A": "Respuesta correcta",
    "B": "Opción falsa 1",
    "C": "Opción falsa 2",
    "D": "Opción falsa 3"
  }
}

🔄 Cómo Extraer del PDF de USCIS:

  1. Abre el PDF: 2025-Civics-Test-128-Questions-and-Answers.pdf
  2. Lee cada pregunta y sus respuestas correctas
  3. Traduce al español usando un traductor confiable (Google Translate, DeepL)
  4. Crea opciones falsas que parezcan cercanas pero no sean correctas
  5. Agrega a us_civics_spanish.json en formato JSON
  6. Recarga la app para ver las nuevas preguntas

⚙️ Consejos Importantes:

  • "correct" DEBE ser "A", "B", "C", o "D"
  • Las opciones se mezclarán automáticamente cada pregunta
  • Asegúrate de JSON válido (sin comas extras o sintaxis incorrecta)
  • Puedes agregar hasta 128 preguntas
  • El examen selecciona 20 aleatoriamente
  • Recarga (F5) la página después de editar el JSON
